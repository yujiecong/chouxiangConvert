# chouxiangConvert
以前仿照网上的抽象生成器写的一个python版本

## Requirement
 * pypinyin
 * jieba
## Demo
```
cmd不支持输出emoji表情，但可以复制出去用，所以不要慌
输入1选择轻度抽象，输入2选择高度抽象,输入3添加抽象字符2
输入需要抽象的句子
例如.我真的就不信了，哪来的这么多的抽象字== 👴💉💧9️⃣8️⃣✉🌶️，🌶️来💧这么多💧抽🐘字真不错，住在山里真不错
```
